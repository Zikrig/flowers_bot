# Быстрый старт для администратора

## Шаг 1: Настройка

1. Узнайте ваш Telegram ID через бота [@userinfobot](https://t.me/userinfobot)
2. Добавьте ваш ID в файл `.env`:
   ```env
   ADMIN_IDS=ваш_telegram_id
   ```
3. Отправьте `/start` вашему боту

## Шаг 2: Работа с заказами

### Когда клиент отправляет квитанцию:

1. **Вы получите уведомление** с фото квитанции и информацией о заказе
2. **Проверьте оплату** на вашем счету
3. **Нажмите кнопку:**
   - ✅ **"Подтвердить оплату"** — если деньги поступили
   - ❌ **"Отклонить"** — если оплата не подтверждена

### После подтверждения:

- ✅ Заказ автоматически добавляется в Google Sheets
- ✅ Создается бланк заказа (Excel файл)
- ✅ Клиент получает подтверждение

## Полезные команды

| Команда | Что делает |
|---------|-----------|
| `/admin_orders` | Показать все заказы |
| `/admin_orders_pending` | Заказы, ожидающие оплаты |
| `/admin_orders_paid` | Оплаченные заказы |
| `/admin_orders_today` | Заказы на сегодня |
| `/admin_order 042` | Информация о заказе №042 |
| `/admin_stats` | Статистика и выручка |

## Примеры использования

### Посмотреть все заказы:
```
/admin_orders
```

### Посмотреть заказы на сегодня:
```
/admin_orders_today
```

### Узнать детали заказа:
```
/admin_order 042
```

### Посмотреть статистику:
```
/admin_stats
```

## Уведомления об отмене

Когда клиент отменяет оплаченный заказ, вы получите:
- Номер заказа
- Номер карты для возврата
- Сумму возврата
- ФИО клиента

**Важно:** Верните средства в течение 24 часов!

## Где найти данные

- **Google Sheets:** Все подтвержденные заказы автоматически добавляются
- **Бланки заказов:** Папка `orders/` (Excel файлы)
- **Локальная база:** Файл `data/orders.json`
